# Buddy Booking Status Page

Uptime monitor and status page for Buddy Booking backend services, powered by [Upptime](https://upptime.js.org).

## Monitored Services

| Service | URL |
|---------|-----|
| Production API | https://business-api.bookwithbuddy.com/health |
| Staging API | https://staging-api.bookwithbuddy.com/health |
| Dev API | https://dev-api.bookwithbuddy.com/health |

## How it works

- GitHub Actions checks each endpoint every 5 minutes
- Results are committed to this repo
- GitHub Pages hosts the status page at https://status.bookwithbuddy.com
